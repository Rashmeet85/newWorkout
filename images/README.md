# Hunter's System — Image Assets

Place your images in this folder with these exact filenames:

## Character Portraits (PNG, transparent background recommended)
Dark mode:
- dark_e.png   → E-rank character
- dark_d.png   → D-rank character
- dark_c.png   → C-rank character
- dark_b.png   → B-rank character
- dark_a.png   → A-rank character
- dark_s.png   → S-rank character
- dark_ss.png  → SS-rank character
- dark_sss.png → SSS-rank character

Light mode:
- light_e.png through light_sss.png (same naming)

## Silhouettes (for locked ranks)

- silhouette_light.png
